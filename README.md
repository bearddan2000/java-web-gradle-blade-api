# java-web-gradle-blade-api

## Description
A POC for blade framework.

## Tech stack
- cors
- middleware
- endpoints
    - get, put, delete

## Docker stack
- gradle:7.6.4-jdk11-alpine

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
